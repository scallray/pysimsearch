The ConcurrentSimIndex Class¶

ConcurrentSimIndex

Wrapper to allow concurrent SimIndex access

Sample usage:

from pysimsearch.sim_index import MemorySimIndex, ConcurrentSimIndex

index = ConcurrentSimIndex(MemorySimIndex())
index.index_urls('http://www.stanford.edu/', 'http://www.berkeley.edu')
print(list(index.query('stanford')))
class pysimsearch.sim_index.ConcurrentSimIndex(sim_index)¶

Proxy to a pysimsearch.sim_index.SimIndex that allows concurrent access.

ConcurrentSimIndex is compatible with the SimIndex interface. We use concurrent.futures to allow some basic concurrency for indexing and querying. In particular, calls to index_urls() are executed in a nonblocking manner.
